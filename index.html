Alura Books

O AluraBooks é um site desenvolvido com a ajuda dos cursos da plataforma Alura, que ensina como criar sites de maneira prática e didática. Durante o processo de montagem do site, foram seguidas várias dicas e orientações dos cursos, que ajudaram a garantir que o site fosse funcional, organizado e fácil de usar.

O principal objetivo do projeto era criar um site simples e intuitivo, para que qualquer pessoa pudesse navegar facilmente e encontrar as informações de que precisava. Para isso, foi utilizado HTML para estruturar o conteúdo, CSS para dar estilo ao site.

Falando do visual do site, a ideia foi criar um design que fosse agradável aos olhos e que tornasse a navegação mais fácil. A escolha das cores foi feita com cuidado, para que o contraste fosse bom e a leitura ficasse clara, sem dificuldades. As cores são suaves, com detalhes em destaque para chamar a atenção do usuário para o conteúdo mais importante, como as imagens dos livros, que são grandes e bem visíveis. A tipografia (ou seja, as fontes usadas no site) também foi escolhida para ser confortável de ler em qualquer tamanho de tela, seja no celular ou no computador.

Outro ponto importante do projeto foi o design responsivo, que significa que o site se adapta a diferentes tamanhos de tela. Isso é muito importante porque hoje as pessoas acessam sites de várias formas: no computador, no tablet ou no celular. Com o design responsivo, o AluraBooks fica bonito e fácil de navegar em qualquer dispositivo. As imagens, o texto e os botões são redimensionados automaticamente, sem que o usuário precise fazer nada, para que tudo se encaixe perfeitamente na tela.

No geral, a construção do site foi um grande aprendizado, pois a plataforma Alura ofereceu uma base excelente para entender como criar um site de qualidade. Cada aula e exercício foi importante para colocar em prática o que foi aprendido e, ao longo do processo, foi possível melhorar o design e a funcionalidade do site. O AluraBooks, portanto, não é apenas um site sobre livros, mas também um projeto que mostra como a aprendizagem e a prática podem resultar em algo útil e bem-feito.

Bruna Doces

Usando as orientações da Alura, o site tem um visual moderno e simples, com cores fortes e vibrantes que chamam atenção para os doces. As cores principais, como o roxo e o rosa, passam a ideia de alegria e sabor, enquanto o logotipo e a letra em negrito dão um toque atual e fácil de ler.

O site foi pensado para ser fácil e agradável de usar. Ele não tem excesso de informações, então o cliente consegue focar nas imagens dos produtos, que são de alta qualidade. Cada detalhe, como os ícones do menu e os botões de compra, foi feito para facilitar a navegação, usando o que a Alura ensina de melhor. Isso ajuda o cliente a encontrar o que procura de forma simples e rápida, seja explorando as categorias ou vendo os favoritos.

Criado para funcionar bem em qualquer dispositivo — computador, celular ou tablet — usando as orientações da Alura. Isso quer dizer que o site se adapta automaticamente ao tamanho da tela, reorganizando o conteúdo para ficar fácil de ler e usar. O menu vira um "menu hambúrguer" (os três risquinhos), o que economiza espaço e deixa o visual limpo. As imagens e os botões se ajustam de acordo com o dispositivo, então o cliente não precisa ficar ampliando ou movendo a tela para navegar.

Feito com base no que a Alura ensina, o site BrunaDoces combina um visual chamativo com uma navegação simples e prática, além de ser adaptado para qualquer tipo de dispositivo. Isso faz dele uma plataforma convidativa e fácil de usar para quem quer explorar as delícias do mundo dos doces.

Flashcards

O Flashcards Colleen Hoover é um site criado para ajudar as pessoas a aprender e revisar temas dos livros da autora Colleen Hoover de um jeito fácil e divertido. Ele foi desenvolvido seguindo as instruções da plataforma Alura e usando como base o site Flashcards, que é conhecido por organizar o conteúdo em cartões de memória. Com isso, o site ajuda os leitores a explorar detalhes e curiosidades dos livros, facilitando o estudo e a memorização.

O design do site é responsivo, ou seja, ele funciona bem em qualquer dispositivo, seja celular, tablet ou computador. As cores foram escolhidas para deixar o site bonito e agradável, combinando com o clima envolvente dos livros de Colleen Hoover. Tons suaves e destaques criam uma experiência visual agradável, deixando o conteúdo claro e fácil de ler.

O site foi feito com HTML, CSS e JavaScript, o que permite interatividade, assim os usuários podem navegar pelos flashcards e aprender de forma dinâmica. O layout foi planejado para ser simples e direto, permitindo que os leitores encontrem o que precisam sem dificuldades. É uma maneira prática de conhecer mais sobre a obra de Colleen Hoover e suas histórias de uma forma rápida e organizada.
